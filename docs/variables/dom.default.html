    渲染虚拟节点

     1、对每一个组件配置项进行递归渲染成虚拟节点
 2、保证每个组件的插槽可用
 3、对于不存在响应式变量的虚拟节点进行静态提升，减少渲染次数
 4、对于组件配置项中的事件进行缓存优化
 5、使用Fragment将多个虚拟节点合并成一个虚拟节点，减少层级结构
 6、支持v-model的简写，且支持多个v-model
 7、支持v-if、v-else-if、v-else等多个指令
 8、支持一个对象有一个作用域来存储上下文
 9、支持对象之间的上下文继承关系，也可以跨对象层级获取上下文
